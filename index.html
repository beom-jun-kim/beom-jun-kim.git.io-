About me

각 회차마다 웹 퍼블리셔가 되기 위해 배웠었던 전체적인 커리큘럼을 담아보았습니다 알맹이가 있는 실무 위주의 공부만 한다라는 목표와 함께 진행하였습니다

1개월차 - html

html의 시맨틱 태그들을 배우고 로그인, 약관, 가입하기 위주의 여러 마크업 실습들을 진행 하였습니다

2개월차 - CSS

CSS언어를 통해 각 레이아웃을 위치시키는 법등을 배웠고 style을 입히는 작업을 하였습니다

3개월차 - 반응형

반응형에 필수적인 미디어쿼리를 통해 grid를 달리하여서 태블릿이나 모바일에서 보여지는 화면 구현을 배웠습니다

4개월차 - Javascript

Javascript의 기본인 함수선언부터 시작하여 클릭 이벤트를 통하여 컨텐츠들을 더욱 동적으로 바꾸는 법까지 배워보았습니다

5개월차 - jQuery

jQuery 라이브러리로 더욱 쉽게 실행시키면서 여러가지 방법으로 컨텐츠를 역동적이게 구현이 되도록 해보았습니다
